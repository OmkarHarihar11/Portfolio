### Overview
A multi-page personal portfolio with immersive 3D animations, RGB neon glow effects, and a dark futuristic theme. Built with React Three Fiber for 3D visuals and smooth page-to-page transitions with animated routes.

### Design & Animation Details
- **Color Palette**: Deep black background + neon RGB (purple `#a855f7`, cyan `#06b6d4`, pink `#ec4899`) cycling gradients
- **3D Engine**: React Three Fiber + React Three Drei for all 3D visuals
- **Page Transitions**: Smooth fade + scale animation between routes
- **Scroll Animations**: Elements animate in as you scroll
- **Mouse Parallax**: 3D elements subtly follow cursor movement
- **Floating Navbar**: Glassmorphism style with RGB-glowing active link indicators
- **Typography**: Large, bold fonts with glowing text effects
